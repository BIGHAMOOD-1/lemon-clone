# 数据清洗程序使用说明

## 功能

从原始数据中提取指定人物的内容，并将所有内容用 `:` 分割后输出到文件。

## 使用方法

```bash
python clean.py <输入文件> <人物名称> [输出文件]
```

### 参数说明

- `输入文件`: 包含原始数据的文件路径
- `人物名称`: 要提取的人物名称（如 "DjjD"）
- `输出文件`: 可选，输出文件路径，默认为 `1.txt`

## 示例

```bash
# 提取 DjjD 的内容，输出到 1.txt
python clean.py input.txt DjjD

# 提取 DjjD 的内容，输出到 custom_output.txt
python clean.py input.txt DjjD custom_output.txt
```

## 数据格式

程序支持以下格式的原始数据：

```
DjjD:

时间: 2025-03-06 19:08:29

内容: @黑田悠斗

提及: 黑田悠斗
```

程序会提取所有匹配人物的"内容"字段，并用 `:` 连接。

## 输出格式

所有提取的内容会用 `:` 分割，例如：

```
内容1:内容2:内容3:内容4
```
